Form

Form HTML

Form digunakan untuk mengirimkan data ke server.

Tag <form> digunakan untuk mendeklarasikan HTML form.

<form>
elemen input
</form>

Text Fields

<input type="text"> mendefinisikan sebuah teks input yang bisa diisi oleh user, inputan akan terlihat oleh user.

<form> Nama : <input type="text" name="nama"> </form>

Password Fields

<input type="password"> mendefinisikan sebuah teks input yang bisa diisi oleh user, inputan tidak akan terlihat oleh user.

<form> Sandi : <input type="password" name="sandi"> </form>

Radio Button

<input type="radio"> mendefinisikan sebuah input pilihan yang bernilai salah satu.

<form>
<input type="radio" name="jenis-kelamin" value="laki-laki" /> laki-laki
<input type="radio" name="jenis-kelamin" value="perempuan" /> perempuan
</form>

Tombol Submit

<input type="checkbox"> mendefinisikan sebuah input pilihan yang bernilai banyak.

<form method="POST" action="http://alamatweb/form.php>
Nama : <input type="text" name="nama" /> 
<input type="submit" value="submit" />
</form>

  • Submit Button digunakan untuk mengirimkan data ke server.
  • Data dikirim ke sebuah halaman yg di deklarasikan pada atribut milik form.
  • <input type="submit"> mendefinisikan input pilihan yang bisa bernilai semuanya/banyak.
