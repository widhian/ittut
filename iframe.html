Iframe

Iframe HTML

  • tag : <iframe>, digunakan untuk memulai iframe.
  • Elemen iframe digunakan untuk menampilkan sebuah halaman web di dalam sebuah halaman web.

Contoh :

<iframe src="http://alamatweb" width="200" height="200" name="iframe1">

  • atribut src merupakan alamat source dimana akan ditampilkan di iframe tersebut.
  • atribut width merupakan lebar dari iframe.
  • atribut height merupakan tinggi dari iframe.
  • atribut name merupakan nama dari iframe.
