Head

Elemen <head> HTML

Elemen <head> adalah sebuah kontainer untuk semua elemen head. Elemen di dalam <head> dapat meliputi script, style sheet, meta, dan masih banyak lagi.

Beberapa yang dapat dimasukkan ke dalam sesi head adalah : <title>, <style>, <meta>, <link>, <script>, <noscript>, dan <base>

Elemen <title> HTML

Elemen <title> mendefiniskan judul dokumen. Elemen ini dugunakan di seluruh dokumen HTML/XHTML

Elemen <title> :

  • Mendefinisikan judul di toolbar browser
  • Menjadi judul dari halaman ketika dijadikan favorit halaman
  • Menampilkan judul di halaman Search Engine

<!DOCTYPE html>
<html>
<head>
<title>Title of the document</title>
</head>

<body>
The content of the document......
</body>

</html>

Elemen <base> HTML

Tag <base> mendefinisikan URL base untuk semua URL relatif di halaman

<head>
<base href="http://sumber/images/" target="_blank">
</head>

Elemen <link> HTML

Tag <link> mendefinisikan hubungan antara dokumen dengan sumber eksternal

<head>
<link rel="stylesheet" type="text/css" href="style.css">
</head>

Elemen <meta> HTML

Tag <meta> menyediakan informasi data

Meta untuk menampilkan keyword halaman web di search engine

<meta name="keywords" content="HTML, CSS, XML, XHTML, JavaScript">

Meta untuk menampilkan deskripsi halaman web di search engine

<meta name="description" content="Free Web tutorials on HTML and CSS">

Meta untuk menampilkan author halaman web di search engine

<meta name="author" content="Hege Refsnes">

Meta untuk merefresh dokumen setiap 30 detik

<meta http-equiv="refresh" content="30">

Elemen <script> HTML

Tag <script> mendefinisikan script client-slide seperti javascript
