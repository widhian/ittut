Tautan

Sintaks Tautan HTML

<a href="http://alamat" target="(target)">Tulisan ini akan di tampilkan</a>

  • <a> adalah tag pembuka dari elemen tautan
  • href mendefinisikan alamat url dari halaman yang akan dituju
  • target merupakan target tautan itu saat dibuka

Target Value HTML

Value Deskripsi
_blank Dokumen akan terbuka di jendela baru atau tab.
_self Dokumen akan terbuka di frame yang sama dengan halaman yang diklik.
_parent Dokumen akan terbuka di parent frame
_top Dokumen akan terbuka di full body dari jendela.
(nama frame) Dokumen akan terbuka di frame tertentu
