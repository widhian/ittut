Tabel

Elemen <table> HTML

<table border="1">
<tr><th> A </th> <th> B </th>/tr>
<tr><td> Jono </td> <td> Budi </td></tr>
<tr><td> Wati </td> <td> Anjas </td></tr></table>

Hasil :

A B
Data A1 Data B1
Data A2 Data B2

Keterangan :

  • Tag pembuka tabel : <table>
  • Tag baris : <tr>
  • Tag kolom : <td>
  • Atribut border digunakan untuk menampilkan ketebalan garis tepi tabel
  • Tag header tabel : <th>, digunakan untuk mendefinisikan kepala tabel
