Format

Tag Text Format HTML

Tag Deskripsi
<b> Untuk membuat text bold (cetak tebal)
<em> Untuk membuat emphasized text (cetak miring)
<i> Mendefiniskan bagian dari text alternatif suara atau mood. Hasilnya adalah tulisan miring.
<small> Membuat text menjadi kecil
<strong> Mendefinisikan text penting. Hasilnya adalah text dengan cetak tebal.
<sup> Mendefinisikan text superscript. Hasilnya tulisan berada di atas.
<sub> Mendefinisikan text subscript. Hasilnya tulisan berada di bawah
<ins> Mendefinisikan inserted text. Hasilnya tulisan di garis bawah.
<del> Mendefinisikan deleted text. Hasilnya tulisan di coret garis.

Tag "Computer Output" HTML

Tag Deskripsi
<code> Mendefiniskan text code computer
<kbd> Mendefinisikan text keyboard
<samp> Mendefinisikan sample code computer
<var> Mendefinisikan variabel. Hasilnya adalah tulisan miring.
<pre> Mendefinisikan preformatted text. Elemen di dalamnya termasuk baris baru dan banyaknya spasi akan ditampilkan apa adanya.

Tag Citation, Quotation, dan Definition HTML

Tag Deskripsi
<abbr> Mendefinisikan akronim.

<abbr title="World Health Organization">WHO</abbr>
<address> Mendefinisikan informasi kontak atau author
<bdo> Mendefinisikan cara membaca arah text.Value "ltr" adalah Left to Right sedangkan value "rtl" adalah Right to Left.

<bdo dir="rtl">Text ini dituliskan dari kanan ke kiri</bdo>
<blockquote> Mendefinisikan text quote dari sumber lain.

<blockquote cite="http://sumber">Isi Quote.</blockquote>
<q> Mendefinisikan quote pendek (inline)
<cite> Mendefinisikan judul dari pekerjaan. Hasil akan dicetak miring.
<dfn> Mendefinisikan sesuatu. Hasil akan dicetak miring
