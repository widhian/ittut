Pendahuluan

Apa itu HTML

HTML merupakan bahasa untuk menampilkan halaman web

  • HTML merupakan singkatan dari Hyper Text Markup Language
  • HTML adalah bahasa markup
  • Bahasa markup merupakan kumpulan dari tag markup
  • Dokumen HTML mempunyai tag HTML dan plain text
  • Dokumen HTML juga disebut sebagai halaman web

Tag HTML

Markup HTML Tag biasa disebut dengan Tag HTML

  • Tag HTML merupakan keyword (nama tag) yang diapit oleh tanda "<" dan ">" seperti <html>
  • Sebagian besar Tag HTML adalah berpasangan, seperti <b> dan </b>
  • Bagian tag pertama di tag yang berpasangan adalah start tag dan tag kedua merupakan end tag
  • End tag dituliskan seperti tag pertama, namun dengan forward slash sebelum nama tag
  • Start dan end tag juga disebut dengan tag pembuka dan tag penutup

<nama tag>konten</nama tag>

Elemen HTML

"Tag HTML" dan "Elemen HTML" sering digunakan untuk mendeskripsikan hal yang sama.

Namun, perbedaannya, Elemen HTML merupakan keseluruhan dimulai di dalam Start Tag dan End Tag

<p>Ini adalah paragraf</p>

Web Browser

Web Browser seperti Google Chrome, IE, Firefox, Safari, dsb digunakan untuk membaca dokumen HTML dan menampilkannya sebagai halaman web.

Browser tidak menampilkan Tag HTML, namun menggunakan tag tersebut untuk merepresentasikan konten halamannya

Struktur Halaman HTML

<html>

<body>

<h1>Heading</h1>

<p>Paragraf.</p>

<p>Paragraf lain.</p>

</body>

</html>

Deklarasi <!DOCTYPE>

Deklarasi <!DOCTYPE> membantu browser untuk menampilkan halaman web dengan benar.

Di dalam web terdapat tipe dokumen yang berbeda-beda, dan browser hanya dapat menampilkan halaman HTML 100% benar jika browser mengetahui tipe HTML dan versinya

Deklarasi Doctype Umum

HTML5

<!DOCTYPE html>

HTML 4.01

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

XHTML 1.0

<DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
